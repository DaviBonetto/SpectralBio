## New Notebooks

This folder contains the new Colab-oriented notebook campaign for the post-review SpectralBio upgrade.

Conventions:

- Notebook names start with a zero-padded execution order prefix.
- Each notebook writes its own results under `New Notebooks/results/<notebook-slug>/`.
- Each runnable code cell should end with a `TERMINEI PODE SEGUIR` message.
- Colab-first notebooks should zip their outputs automatically and trigger a download when possible.
